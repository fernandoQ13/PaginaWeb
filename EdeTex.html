Etiquetas de textos y encabezado

Las siguientes etiquetas que sele presentaran son etiquetas de texto y de encabezado
las cuales sirven para mostrar diferentes tamaños de letras, saltos de líneas , resaltado
de texto, subrayado de línea , entre otro.
A continuación se le mostraran un grupo diferentes de etiqueta para poner en practica.

  • Etiqueta strong y b
  • Etiqueta i,em,,cite
  • Etiqueta br
  • Etiqueta mark
  • Etiqueta big y small
  • Etiqueta p
  • Etiqueta s y del
  • Etiqueta u
  • regresar

Las etiquetas < strong> ,< b> : Permite poner las letras en negritas.
Ejemplo : < strong>texto desea < /strong>, < b>texto deseado< /b>.
etiqueta strong y etiqueta b.

Etiquetas < i>,< em>,< cite> : Define un tipo de letra distinto.
Ejemplo: < i>texto deseado< /i>, < em>texto deseado< /em>,< cite>texto deseado< /cite>.

etiqueta i,etiqueta em ,etiqueta cite
como veran las tres dan el mismo funcionamiento.

Etiquete < br>: Realisa un salto de linea,se debe de colocar justo en el momento
que desea dar un salto de linea en el texto,solo debe de escribir < br>.

Etiquete mark : Define un resaltado.
ejemplo : < mark>texto desea< /mark>.
etiqueta mark

Etiquetas big y small : Ambas etiquetas sirven para definir un tamaño de letra,una
agranda el tipo de letra y la otra lo reduce.
Ejemplo : < small>texto< /small> ,< mark>texto< /mark>.
Etiqueta small ,"texto sin etiqueta", etiqueta big.
